Máster en Desarrollo Full Stack

Este máster está diseñado para quienes desean aprender a crear aplicaciones web completas, tanto en el lado del cliente (frontend) como en el lado del servidor (backend). Aprenderás tecnologías como HTML, CSS, JavaScript, Node.js, y bases de datos como MongoDB y SQL.

¿Qué aprenderás?

  • Fundamentos del desarrollo web (HTML, CSS, JavaScript)
  • Desarrollo Backend con Node.js y Express
  • Gestión de bases de datos (MongoDB y SQL)
  • Frameworks frontend como React
  • Despliegue de aplicaciones en la nube
